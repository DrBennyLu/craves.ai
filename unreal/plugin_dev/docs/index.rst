Welcome to unrealcv_research_scripts's documentation!
=====================================================

The folder contains scripts for several different research projects and some common scripts for performing common tasks.

- diva
- HMC
- weight_analysis

The goal of this repo demonstrating interesting applications of synthetic data.

In order to make synthetic data useful, it requires to generate a large dataset, do data preparation to convert to meaningful data format, then train or test a model.
